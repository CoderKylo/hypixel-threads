heyo, this took me a few weekends and some weekdays to finally learn. i still haven’t learned basic grammar rules and sentence structure in korean, but, with the help of a study set on quizlet, i learned the 24 basic characters of the korean alphabet, translated into english.

when i first started the language around a month or 2 ago, i was expecting it to be really hard. but to my surprise, it wasn’t actually that complicated. i first learned that there are 24 characters with 10 vowels and 14 consonants. it was fun learning the first few vowels like ㅏ (a) and ㅓ(eo).

ㅂ = b (bucket)

ㅈ = j (jogger)

ㄷ = d

ㄱ = g

ㅅ = s

ㅛ = yo

ㅕ = yeo

ㅑ = ya

ㅐ = ae

ㅔ = e

ㅁ = m

ㄴ = n

ㅇ = ng (silent)

ㄹ = l (or r i’m not completely sure about that one)

ㅎ = h

ㅗ = o

ㅓ = eo

ㅏ = a

ㅣ = i

ㅋ = k

ㅌ = t

ㅊ = ch

ㅍ = p

ㅠ = yu

ㅜ = u

ㅡ = eu

when i started learning about the silent character, ㅇ, i learned how there had to be a filler consonant if there was only 1 vowel and no consonant.

ex:
ㅏ (wrong)
아 (right)

then i got into the placements of each character in a block. copied from my notes:

initials - always a consonant/positioned on top
medials - always a vowel/positioned in middle
finals - always consonants/positioned below

ex: 은 (eon - pronounced as “oon”)

after i learned this, i finally understood how to write a korean block which got me pretty hyped. but i still didn’t know 90% of the alphabet. so that’s where quizlet came in. i found a study set of the alphabet with some extra confusing parts, but it had the alphabet so it was ok i guess. after a few hours or so, i can recognize all letters of the korean alphabet and translate them into english.

can i read korean? no
can i write korean? partially, but no
can i speak korean? no
can i recognize characters in korean? yes

next, i translated the korean characters in a wiki page about bananas into english letters



(obviously missed the hyphens)

the only thing i’m having issues with is the difference between L and R in korean. you can tell the google translate changed most of the R’s into L’s. i’ll look further into that tomorrow and see where i went wrong.

maybe the L is pronounced as an R?

I’M GETTING THERE. ONCE I LEARN KOREAN (maybe a few years), I’M GONNA WATCH SQUID GAME WITHOUT SUBTITLES FOR THE EXTRA SWAG.
