If there is one thing that hypixel is not that great at then it would be social media. They don't try to put much effort into social media at all. It's a bit sad, because if all of those things had more effort (Besides the forums, teddy has that covered) then I feel like more people would find out about the server and it would encourage their community to use social media for the server as well.
