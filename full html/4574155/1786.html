On second thought, lets start troubleshooting before I get your response.

1. Press the start key, search for cmd and press run as administrator. Type the following command "sfc /scannow" and wait on it to finish, then restart your system.

2. Once you have booted back into Windows, press the start key and search for cmd and press run as administrator. Type the following command "DISM /Online /Cleanup-Image /CheckHealth" and wait on it to finish, then restart your system.

3. These commands will scan your computer for corrupted files and attempt to restore them, which might help if it's a corrupted system file causing this issue.

4. Try updating your drivers and operating system

5. Check task manager when this happens and filter by name, is it a program that shows up for this?

6. What was the last thing you did before this started happening?

7. Perhaps try running a antivirus scan, you can install something like Malware Bytes and scan it, or just scan with windows defender or something. (I would reccomend you uninstall the antivirus afterwards as they slow down your computer. Btw, PUPs are potientially unwanted programs.)

8. What did you first do/install right before this started happening?
