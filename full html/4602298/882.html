Thought he was better but guess not.

Here's my take on it and best of all this is the type of stuff I have experience with(5 years ish),
-Video is overly hyped

-The Tab gui is a standard easy to copy paste click gui found in most hacked clients however it looks worse too. Heroes click gui is a lot nicer and is free.

-Math or rendering is done wrong in the tab gui too lol and is easy to figure out what is wrong.

You can see that where the slider starts at is x = 0 and it renders lets say 2 pixel to the right to complete the bar.
So x1 = 0, x2 = 2
Now What Happens is when you move the slider, it only changes x1 thus creating a gap. So say you move it 5 units to the right.
x1 = 5 and x2 = 2
That's why a gap of 2 pixels is still there.
This is simply something that should have not made it into a production launch.

-Not able to drag other mod statuses, another reason to believe click gui is copy pasted as the dragging code is nearly identical.

-Colour of statuses is not affected

-The vid also shows more ways other stuff was copy pasted too.

Overall, crappy client and I actually think I could do better. I also don't wanna bother taking a look at the source code but wont be surprised if it's copy pasted from a yt video.
