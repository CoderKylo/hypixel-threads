Most likely newer version files of Lunar is Encrypted...
But I have seen a really old version of Lunar that was decrypted by someone.

Download : Click Me to Download

After you download that version you can do Extract wit WinRAR or 7Zip...

It will look something like this after extracting...

After that its pretty easy to change what you want...
For example
Go to C:\Users\Ganidhu\Desktop\Lunar\LC\assets\lunar\logo (Where ever you extracted the file)
After that you can replace these Images with what ever Logo you want.

Make sure u put like logo-100x100.png as ur logo name and drag to this folder and replace your logo with Lunar Logo.


Also You can go this Lunar\LC\assets\lunar\backgrounds path and change this image too...

After you make all the changes, Go all the way back to C:\Users\Ganidhu\Desktop\Lunar\LC



And Select all the files by doing Ctrl+A
Now right click and select Add to Archive


Now put this select these options like this.
[Archive Format: ZIP]

Then press OK. Now it should Archive all the files.

After it is Complete... Scroll all the way down untill you find the ZIP file.

Now click on it and do Ctrl+X (Cut) now go to C:\Users\Ganidhu\Desktop\Lunar Folder and do Ctrl+V (Paste)

Now select the Zip file and press F2 Now rename it to LC.Jar

Next it should give you a warning Saying if you change file name, extension file might be unusabal... Just press Yes.

Now go to your first downloaded Folder.

Now Copy and Paste the modified LC.Jar file...

Now Go pack by clickign this button.

Now Copy this file

And do Windows+R (Windows RUN) and type %AppData%/.minecraft

Go to Versions folder. And paste the copied folder

Now open up Minecraft Launcher
Go to Installations

Press New Installation.

Put what ever name you want And select the version Release LC and press Create.


Now click on Play and it should work.
