Guys, guys, GUYS! I found what I've been after for MONTHS.

Quick backstory, I actually used to be a server admin in a super small Minecraft Network, and I had control over the Public SMP there, meaning configs, plugins, console, it was under my control.

Let's not talk about the fact that I killed the whole SMP. Why? Cause I know how bad Minecraft Servers are optimized, and even with 6GB Ram, nice processor and Paper, the server still lagged sometimes. As such, I was always finding ways to optimize the server to new highs. So one day, not even a Nucler Bomb could take that thing down.

I may have accidentally installed a plug in to pregen chunks, and I didn't realise the host we were using at the time had a 20GB Storage limit, for some reason... And the server auto-shutdown to prevent corruption to the files.

This was all because I wanted to make sure the players wouldn't experience any lag while playing on the server.

So, when it came to my attention 1 year ago Minecraft just wasn't able to have Multithreaded processing, I was furious. Really? One of the techniques all servers use to process all their data, is non-existent in MC? Yeah, so after I heard about that, I said to myself there had to be a way. But it is impossible, right?







As of today, I have officially found, a Multithreaded solution for MC Servers. I'm talking about.. Minestom.

From Minestom.net:
"A Minecraft server implementation,
open-source and without any code from Mojang."

That is right. No GUI, No AI, No World Gen, No Mojang. Then why does it even matter.

For 1, its Multithreaded, so you can have a lot more players and processing being done in parallel. And 2nd, No Mojang, means you have control over everything, not building on top of Mojang's implementation. Meaning, performance is unmatched. If you want those features, you can code them yourself or get them from the, kind of extensive Extension and Libraries section on their website.

Now, I know this is Off-Topic, but could Hypixel implement this? The answer is, unlikely. Hypixel has money, dedication, developers, everything. But that would mean needing to rewrite all of their plugins. Because while yes, now you have access to Multithreaded Processing, Bukkit, Spigot, Paper and others don't, and the code is super different, meaning, you guessed it, No way to use those plugins while running Minestom.

Hypixel could though. If they were willing to accept these limitations. + imagine Minestom added transfer packets oh yeah, that would be cool.

So, I finally can say to everyone that says "Only Mojang can add Multithreaded Processing and other things" a simple, quick and power, "Nope, we don't need to wait for those bastards to implement anything."

Tell me what you think. Have you heard of this before?

Best regards, Nyxw
