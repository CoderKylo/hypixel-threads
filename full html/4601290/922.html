So the first mob revealed is the glare
A bush like creature that tells you where mobs can spawn
Seemingly spawns in caves
Pros: cute
Living bush
Could be used for redstone
Cons: functionally unless they remove f3, f3 is better
Especially since there’s a rumor that Mojang making mobs spawn in only light level 0
Second mob revealed:
The allay
A blue friendly flying vex
Looks to spawn in the flower forest
Pros: cute as well
Will pick up items, useful for sorters/farms
Cons: might not do it in the ideal way we imagined it
Does not dupe items
Copper golem-third mob
A mob you create yourself
Pros: cool cute baby version
Can make statues when oxidized? If it stops working judging by the trailer
Made of copper a common resource
Cons: not much use functionally except for rng machines or minigames

Imo
My vote would go
Allay
Copper golem
