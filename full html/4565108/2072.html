There a lot of boosting FPS mods and clients for minecraft but all of them dont give you the maximum fps this is what this guide is all about! increasing you're fps to MAX. hope you find this thread useful!

Sooo you're stuck at 30fps well let me fix that for you.
if you want the maximin possible FPS continue reading!
Its better if you start doing the basics of boosting you're fps, the first step to increasing you're fps is OPTIFINE yes its essential as it gives you more settings about you're graphics but id recommend getting client as it has optifine built in and more fps boosting mods. (optifine.net) (https://www.lunarclient.com/download/)
Next up we want to get the best POSSIBLE settings for optifine don't worry lunar people lunar client has optifine pre-installed same with BLC client.
Here are the tweaks (https://gamertweak.com/best-optifine-settings-fps/).
Next up is HONE, hone is a very good software to clean up windows and boost fps but before you use hone i highly recommend creating a system restore to do that read this guide (https://support.microsoft.com/en-us...re-point-77e02e2a-3298-c869-9974-ef5658ea3be9) Hone discord (discord.gg/hone)
Then hone will have instructions in there discord server to apply hone and boost fps if you are confused don't be afraid to open a ticket!
Still not getting enough FPS don't worry i have more steps!!
Next is windows decrapifier it removes ALL bloatware from windows and increases FPS, download windows decrapifier (https://github.com/n1snt/Windows-Decrapifier) all credit goes to n1snt for this script.
Once downloaded extract the files to you're downloads or any where you like to keep it.
Then, Right click on decrapify.ps1 and click "edit" it will open up notepad press CTRL + A to select all and press CTRL + C to copy it.
After that right click on the windows start button and click Windows PowerShell (admin), once open press CTRL + V to paste the code and hit enter.
The script should start to delete all the bloatware to you're computer after its done restart you're computer.
Thats it you're done Congrats! if this doesn't work install sideload linux need help doing this DM me and i will made a guide on how to dualboot linux and windows 10 or 11
Heres one last step, thanks to the_gold_knight for the recommendation to add this.
The most common way to boosting you're fps is FPS packs fps boosting packs do work and can get you a extra 20fps or so.
Here are some videos that have great fps packs.
all credit goes to the owner of these scripts and videos.
if you want to support me please like this post!
(if this post gets 10 likes i will make a guide on how to dual boot linux or just dm me and ill get to work have a great day )
(AM NOT GOING TO MAKE A OVERCLOCKING GUIDE AS IT CAN RUIN YOU'RE CPU/GPU AND DEGRADE LIFETIME SO DONT ASK ME)

#1 Find a distro right for you, I'll explain with Linux Mint. (Ubuntu based)
#2 Go to https://linuxmint.com/ and download it through Harvard cause it's the fastest server.
#3 Download rufus at https://rufus.ie/en/ (grab a flash drive)
#4 With rufus open, select your .iso and use these settings, everything else default.


5. Start!
6. Restart your computer and enter the BIOS, disable secureboot and select the flashdrive as your boot drive. It should be f2, f10, f8 or del you need to mash to get into your BIOS.
7. Welcome to linux mint, try it and if you like it install it.
8. If you would like to keep windows, press boot alongside Windows Boot Manager (which allows you to dual boot both operating systems) otherwise just clean install.
9. Finish the install
10. Enjoy Linux Mint
